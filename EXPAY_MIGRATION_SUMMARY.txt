🔗 WEBHOOK FLOW (Automatic)
=============================

ExPay (after user pays) →
  POST /api/wallet/webhook/ with:
    {
      "event": "PAYMENT_SUCCESS",
      "orderId": "...",
      "status": "SUCCESS",
      "amount": "100.00",
      "utr": "..."
    }

Django webhook handler:
  1. Verifies HMAC-SHA256 signature
  2. Finds Order by smile_order_id
  3. Marks order as COMPLETED
  4. Credits wallet (for wallet top-ups)
  5. Triggers Smile.one fulfillment
  6. Returns HTTP 200

You don't need to do anything — the webhook is fully implemented! ✅
